Grid System

Explanation: display: grid makes a 'div', a grid container, grid is made up of grid container and its direct children as grid items.

Points to understand

  1. We have column axis and row axis, column as y axis and row as x axis.
  2. This axis are not interchangeable as we can do in flex box.
  3. There is grid line in grid layout which is imaginary line which separates rows and columns,
  4. They are numbered automatically for columns and rows. starting from 1 to n.
  5. Space between columns are called column gutter and space between rows are called as row gutter.
  6. Space between two grid lines is called a track. if the track is horizontal it is called row. if it is vertical it is called column.
  7. Area between two horizontal and two vertical grid line is called grid area making a square or rectangle.
  8. Area between two adjacent grid line is called grid cell.

Implicit and explicit grids


Explanation:
When we define grid with grid-template-rows and grid-template-columns in this case we defined 2 x 2 grid i.e. 2 rows and 2 columns, but the grid items are more than 4 items hence those which comes under 2x2 grid defined by us are called explicit grid, whereas grids which are created and are positioned below these grids are called implicit grids. we can style these implicit grids by using grid-auto-rows: 100px this will give height to the implicit grid row.

Alignment of grid items.


Explanation: align-items: start | end | center | stretch;
If we use align-items: center the grid items inside grid area will get centered vertically same as that in flexbox. if we use align-items: start the grid items will be placed at start inside grid area as same as that in flexbox. if we use align-items: end the grid items will be placed at bottom inside grid area as same as that in flexbox. if we use align-items: stretch the grid items will get stretched inside grid area as same as that in flexbox, this is default behavior.

Using max-content and min-content instead of 1fr in grid-template-rows or grid-template-columns solution: using minmax

grid-template-rows: repeat(2, minmax(150px, min-content)) using minmax function it means the minimum height of row must be 150px and maximum height can be depending on content. similarly we can use max-content with rows. Note:minmax function we are using is to define for only one column or row.

the fr unit also behaves similar like min-content, it takes equal width if multiple columns or rows are there but when the rows or columns are shrinked the fr unit shrinks as well, but stops when it reaches the space where the word can be fitted inside grid
